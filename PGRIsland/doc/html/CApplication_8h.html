Passive mouse motion callback for GLUT's glutPassiveMotionFunc

Checks if mouse cursor is not in the center of screen. Is so, calculate delta in x and y axis. For x delta pan the camera and y delta tilt the camera.

After all mouse movement events were handled, recenter the mouse cursor to window center.

Parameters
mouseX- mouse cursor X coordinate in window space
mouseY- mouse cursor Y coordinate in window space
